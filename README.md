Deskripsi Singkat Tentang Aplikasi
Aditya'sWeb merupakan aplikasi blog full-stack berbasis Laravel yang dikembangkan sebagai proyek pemenuhan Ujian Akhir Semester mahasiswa Teknik Informatika UIN Maulana Malik Ibrahim Malang. Aplikasi ini menghadirkan sistem Content Management System (CMS) dinamis dengan antarmuka yang mengimplementasikan desain UI modern memadukan Neumorphism dan Glassmorphism via Tailwind CSS. Platform ini difokuskan pada lima pilar kategori informatika: Pemrograman, Basis Data, Internet of Things (IoT), Rekayasa Perangkat Lunak, dan Kecerdasan Buatan (AI), menjadikannya repositori pengetahuan teknis yang interaktif dan responsif.
